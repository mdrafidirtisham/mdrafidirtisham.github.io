PROJECT 01

Premier League Player Performance Analysis

BNM862 — Performance Analytics · Aston University · Grade: 76%

PIM-DEA SFA Stata Translog COLS / MOLS
⚽

Evaluated the technical efficiency of 499 Premier League players using Data Envelopment Analysis (CRS & VRS orientations) and three econometric frontier models: COLS, MOLS, and SFA.

After removing goalkeepers and cleaning outliers, ran PIM-DEA with both univariate (1 input / 1 output) and bivariate (1 input / 2 outputs) specifications. Mohammad Salah was the sole efficient player on the CRS frontier. Chose two underperforming forwards — Che Adams and Michail Antonio — for deep-dive benchmarking.

Selected a Translog production function over Cobb-Douglas following diagnostic testing (RESET test confirmed omitted variables in C-D). SFA-EXP-BC model returned the highest and most realistic efficiency estimates.

  • xG and xAG had strongest correlations with Goals+Assists (r = 0.904 and 0.836)
  • SFA noise variance exceeded inefficiency variance — confirms stochastic noise in football data
  • VRS models more appropriate than CRS — smaller DMUs treated more fairly on frontier
  • Translog 2nd-order terms jointly significant (F = 24.67, p < 0.001)

PROJECT 05

UK COVID-19 Vaccination Programme Analytics

Software Analytics (BNM861) · Aston University · Grade: 72%

OLS Regression Python / Pandas Chi-Square Test T-Test Bokeh / Matplotlib
💉

End-to-end exploratory and inferential analysis of the UK Government COVID-19 vaccination dataset (early 2021 – mid 2022), covering First, Second, and Third dose rollout across all UK regions (England, Scotland, Wales, Northern Ireland).

Conducted a full 8-step analytical pipeline in Python: descriptive statistics, missing value treatment (fillna(0)), histogram and scatter plot visualisations, contingency table construction, Chi-square independence test, subset t-tests, grouped aggregations, and finally an OLS multiple regression model using Statsmodels and Bokeh.

The final regression model predicted ThirdDose uptake from First and Second doses, working day, region, and month — achieving an R² improvement from ~0.3 (simple) to ~0.5 (multiple). All p-values on predictors were statistically significant.

  • First dose distribution positively skewed — peak at ~2,500 doses; majority between 0–5,000
  • Strong positive correlation between First and Second dose uptake (scatter plot confirms linear relationship)
  • Chi-square test: areaName and day of week are independent (p > 0.05 — fail to reject H₀)
  • T-test on England subset: no significant difference between First and Second dose means (p = 0.1246)
  • Q3 showed highest median First Dose uptake (~15,000); Q1/Q2/Q4 all below 1,000 median
  • OLS model: ThirdDose = −1.24×10⁵ + 3.84·FirstDose + 8.63·SecondDose + 1,482·WorkingDay + region dummies + 5,912·month
